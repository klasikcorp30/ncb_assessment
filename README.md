
# Black Jack 0.0.1

A blackjack single player game written Javascript.



## Run Locally


Clone project

```bash
  git clone https://github.com/klasikcorp30/ncb_assessment.git
```

Go to the project directory

```bash
  cd ncb_assessment
```

Install dependencies

```bash
  npm install
```

Start the game

```bash
  npm run game
```

Run Test

```bash
  npm run test
```
